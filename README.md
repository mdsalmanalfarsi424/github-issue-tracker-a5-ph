1️⃣ **What is the difference between var, let, and const?**                                                                                                                                         


var: It is function-scoped. It can be re-declared and updated. It also supports hoisting.

let: It is block-scoped. It can be updated but cannot be re-declared within the same scope.

const: It is block-scoped. It cannot be updated or re-declared. It is used for variables that remain constant.                                                                                                                                                     




                                                                     
2️⃣ **What is the spread operator (...)?**                                                                                                                                                         
Spread Operator: It is represented by three dots (...). It allows an iterable (like an array or object) to be expanded into individual elements. It is commonly used for copying or merging arrays and objects.



3️⃣ **What is the difference between map(), filter(), and forEach()?**                                                                                                                                 
map(): Iterates through an array and returns a new array with modified elements.

filter(): Iterates through an array and returns a new array containing only the elements that pass a specific condition.

forEach(): Iterates through an array but does not return anything. It is used for executing a function on each element.


4️⃣ **What is an arrow function?**                                                                                                                                                          
Arrow Function: It is a shorter and more modern way to write functions in JavaScript. It uses the => syntax and does not have its own this context, making it cleaner and easier to read.



5️⃣ **What are template literals?**                                                                                                                                                             
Template Literals: It is a way to create strings using backticks (`). It allows you to easily embed variables or expressions inside the string using the ${} syntax.
